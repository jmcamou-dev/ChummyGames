1. Install Python 

2. In ChummyGames folder open cmd

3. Run this command to create a python virtual environment 
python -m venv venv

4. Run this command to activate the virtual environment
.\venv\Scripts\activate

5. Once activated you can pip install any dependents.
The game folders (SecretHitler) have InstallDependents.txt

6. Once the virtual environment is created and the dependents are 
installed you can now go into the individual game folders (SecretHitler)
and click and run the run_game.bat which will activate you virtual environment
and run app.py
